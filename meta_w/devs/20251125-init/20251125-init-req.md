## Objetivo ##
Recepción y envío de mensajes de whatsapp a traves de la integración con las API (Cloud) que ofrece meta.

## Supuestos ##
- Servicio
El sistema será multi-empresa, es decir que centralizará las comunicaciones de aplicaciones de varias empresas que podrán acceder via api a los mensajes recibidos por su flota de celulares y a su vez podrán postear via api tambien los mensajes que desean enviar a sus propios contactos, indicando el celular de envío.
Para lograr esto deberá contar con un maestro de celulares donde se identifica el numero, la empresa a la que pertenece, el id de meta, el estado y la fecha del estado para saber si está activo.
Cada empresa utilizará los endpoints de la aplicación con un "id_empresa" que le dará acceso a los mensajes recibidos de los celulares que tiene asignado y también a enviar mensajes desde algunos de ellos.
- Plataforma
    + base de datos : sqlpostgres
    + fastapi
    + frontend: react + shadcn admin kit
- Entorno Desarrollo
    + db: local (sqlpostgres)
    + endpoints : fastapi (ngrok)
    + frontend : local
- Entorno Produccion
    + db: Neon
    + endpoints: fastapi (gcp cloud run )
    + frontend : vercel
