# Ejemplos CRUD frontend

Estos archivos son ejemplos de referencia para el patron CRUD frontend.
No se usan en runtime; sirven para copiar patrones de UI.

- list.example.tsx: listado con ResponsiveDataTable (cards responsive)
- form.example.tsx: formulario con wrappers compactos y secciones
- model.example.ts: estructura de model con constants y helpers
